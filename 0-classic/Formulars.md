## Arithmetic Progression (AP)
1. a_n = a_1 + (n - 1) * d
2. S_n = n/2 * (2 * a + (n - 1) * d)
3. ArithmeticMean  = S_n / n

## Geometric Progression (GP)
1. a_n = a_1 * r^(n-1)
2. S_n = a_1 * (r^n - 1) / (r - 1), r > 1
3. S_n = a_1 * (1 - r^n) / (1 - r), r < 1
4. S_infinite = a_1 / (1 - r), r < 1
5. GeometricMean = (a_1 * a_2 ... * a_n-1 * a_n)^(1/n)
